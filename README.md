## Installation

```sh
  package.json scripts
   "scripts": {
    "dev": "next dev",
    "build": "next build",
    "start": "next start",
    "lint": "next lint",
    "format": "prettier -w .",
    "format:check": "prettier -c ."
  },

```

# to install all dependencies

```
yarn install
```

# run dev environment

```
yarn dev
```

# build apps and run apps

```
yarn build && yarn start
```
